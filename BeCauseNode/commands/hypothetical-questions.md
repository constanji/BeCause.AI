## User Prompt Template

```text
请基于以下语义模型生成用户可能提出的不同问题。
语义模型: {{ semantic_model }}
```

## 使用说明

1. **目的**：基于语义模型生成潜在的用户问题，用于增强向量检索
2. **输出格式**：每行一个问题，不包含标签或其他内容
3. **生成要求**：
   - 使用多样化的表达和业务术语
   - 包含不同的句子结构
   - 充分考虑实体、维度、度量和整体语义模型的描述
4. **风格参考**：模仿提供的示例问题的句子结构和风格

## 注意事项

- 生成的问题应该多样化，覆盖语义模型的不同方面
- 问题应该使用业务术语，符合实际用户查询习惯
- 输出格式必须严格：每行一个问题，无其他内容
- 用于生成假设性问题以增强向量检索效果（Hypothetical Document Embeddings, HyDE）
